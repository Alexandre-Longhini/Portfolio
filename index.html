Habilidades

Python

Python é uma linguagem de programação de alto nível, interpretada de script, imperativa, orientada a objetos, funcional, de tipagem dinâmica e forte. Foi lançada por Guido van Rossum em 1991.

FTK Imager

O FTK Imager é uma ferramenta gratuita de visualização de dados e geração de imagens usada para adquirir evidências eletrônicas de maneira forense, criando cópias de dados de computador sem fazer alterações nas evidências originais.

Cellebrite

Cellebrite é uma ferramenta de análise forense digital usada para extrair, decodificar e analisar dados de dispositivos móveis. Ela permite acessar informações como mensagens, chamadas, mídias, aplicativos e dados apagados, auxiliando em investigações criminais, auditorias e perícias digitais.

Nessus

Nessus é uma ferramenta de varredura de vulnerabilidades. Ela identifica falhas em sistemas, redes e aplicações, como configurações incorretas, portas abertas e softwares desatualizados, ajudando a avaliar riscos, priorizar correções e fortalecer a proteção contra ataques.

HTML5

HTML5 é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software. É a quinta versão da linguagem HTML.

CSS3

CSS3 é a terceira mais nova versão das famosas Cascading Style Sheets, pela qual se define estilos para um projeto web. Com efeitos de transição, imagem, imagem de fundo/background e outros, pode-se criar estilos únicos para seus projetos web, alterando diversos aspectos de design no layout da página.

Word

Microsoft Word é um editor de texto da Microsoft, usado para criar, editar e formatar documentos. Ele oferece recursos como correção ortográfica, inserção de imagens, tabelas e gráficos, além de ferramentas de colaboração e exportação em diversos formatos.

Excel

Microsoft Excel é um software de planilhas eletrônicas da Microsoft, usado para organizar, calcular e analisar dados. Ele permite criar tabelas, gráficos, fórmulas e funções automatizadas, sendo amplamente utilizado em áreas como finanças, gestão, estatística e planejamento.
